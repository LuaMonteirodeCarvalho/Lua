📊 Dashboard de Vendas em Excel
Este é um dashboard simples e funcional desenvolvido em Excel para análise de vendas. Ele permite acompanhar os principais indicadores de desempenho de forma visual e intuitiva.

🎯 Missão
Apresentar de forma visual o total das vendas, o resultado de vendas por produto e o total de descontos concedidos no período, permitindo que a gestão de negócios avalie se o resultado está dentro da meta. A partir disso, é possível traçar planos de ação eficazes para atingir os objetivos estratégicos da Companhia.

🚀 Funcionalidades
Visualização de métricas como:

Total de vendas

Vendas por produto

Total de descontos concedidos

🛠️ Tecnologias Utilizadas
Microsoft Excel (com Tabelas Dinâmicas, Gráficos e Segmentações)

💡 Como Usar
Baixe o arquivo dashboard_vendas.xlsx deste repositório.

Abra o arquivo no Microsoft Excel (recomendado: versão 2016 ou superior).
